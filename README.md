# Pokedex for iOS

Pokedex for iOS is a very simple app based on HTTP Requests.

# Features

- Collection Views
- Alamofire
- [PokeAPI](http://pokeapi.co)

# Version
Developed and Tested on iOS 10, Xcode 8 and Swift 3
